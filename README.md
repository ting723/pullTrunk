## 作用: 从SVN的Test目录复制到trunk目录

>示例
*http://svn.demo.cn/demo/demo/code/test/demo1*  
Copy  
*http://svn.demo.cn/demo/demo/code/trunk/demo1*

## 基本功能

* SVN copy 功能
* log 输出到JtextArea中

## 技能点

* svnkit包使用
* swing的基本使用
* swing JTextArea滚动
* swing高级布局  GridBagLayout 练习
* log4j2 配置和灵活使用
* log4j2 日志输出到非文件(如控件中)
* yaml的基本读取
* swing MessageBox自定义
* AES加密和解密
* java读取系统信息(如cpu等)
